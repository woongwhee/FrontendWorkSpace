jQuery 개요

* jQuery란?

기존에 자바스크립트 언어 만으로 구현하기에
복잡했던 구문 작성들을 간소화 하기 위해 "존레식" 에 의해 개발된
Javascript기반 라이브러리(유용한 기능들 모음집)
즉 , Javascript 언어로 유용한 함수나 내용들이 이미 만들어져 있는 모음집이라고 생각하면됨.
복잡한 자바스크립트 구문을 보다 짧고 쉽게 개발가능
(적은양의 코드로 빠르고 풍부한 기능 제공)

-Dom 요소와 관련된 스크립트를 보다 쉽게 구현할 수 있다.
-Ajax(비동기통신), 이벤트처리 등 폭넓게 지원한다.
-jQuery와 관련된 확장된 플러그인, 오픈 API등을 지원한다
(차트, 슬라이드, 애니메이션 등등)

*jQuery 라이브러리 연결 방법

head태그 내 script 태그로 jQuery관련 .js파일 가져오기 단 , 오프라인/온라인방식이 있음
-jQuery 공식 사이트 : 공식사이트 1. jquery 라이브러리 직접 다운로드 받은 후 경로지정(오프라인 방식)
-.js파일을 내가 직접 다운로드 받아 저장 한 후 해당 지정된 파일 경로를 작성하는 방법
2. CDN(content delievery network)를 통한 연결(온라인 방식)
-라이브러리를 제공하는 url을 제시해서 파일 경로를 지정한다.

*jQuery구문 작성하기

jQuery구문으로 Dom요소를 다루는 구문을 기본적으로 작성할 것이기 때문에
jQuert 명려은 문서상 요소들이 다 로드(만들어지고 난 후) 된 후에 실행되어야 한다,

-순수 자바 스크립트에서 해당내용은 
        window.onload=function(){
        실행내용
        }
        onload함수 페이지가 로딩되면서 자동으로 실행되는 함수
        단javascript에서 window.onload구문은 해당 문서의
        요소들도 다 로드되고 뿐만 아니라 연동되어있는 외부문서들도 다 가지고 온 후에
        실행됨, 그리고 window.onload는 해당 문서에서 단한번만 작성 간으.
jQuert에서의 해당내용은 3가지 구문으로 제시된다 
        1)jQuery(document).ready(function(){tlfgodsodyd})
        2)$(document).ready(function(){})
        2)$(function(){})
Jquert에서의 해당 function은 dom요소들만 다 로드되면 바로 실행됨. 여러번 기술가능. 갯수 제한 없음.
