목록 관련 스타일(list)

list-style-type: 불릿기호를 변경해줄때 사용.

        * 순서없는 목록(ul)
        선택자{
            list-style-type: disc/circle/square/none;
        }                    원/빈원/사각형/없음
        * 순서있는 목록(ol)
        선택자{
            list-style-type:decimal/decmal-leading-zero;  숫자,0숫자
                            lower-alpha/upper-alpha; 알파벳 대소문자
                            lower-roman/upper-roman;로마자 대소문자
        }

순서없는 목록

  • HTML
  • CSS
  • JavaScript
  • JQuery

순서있는목록

  1. HTML
  2. CSS
  3. JavaScript
  4. JQuery

list-style-position : 불릿기호의 위치를 조정할때 사용

list-style-image:불릿기호로 이미지를 적용시키고자 할때 사용.

        선택자{
            list-style-position: inside /outside;
            list-style-image : url(적용시키고자하는 이미지의 경로);
        }
