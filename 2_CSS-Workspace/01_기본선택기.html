CSS 기본 선택자

선택자란?

특정 html요소를 선택하고자 할때 사용하는 기능
해당요소를 선택해서 원하는 "스타일"과 기능을 적용시킬 수 있음.

1. 모든 선택자 (*)

모든요소들을 다 선택하고자 할때 사용.

*{
            스타일 속성: 원하는 값
            스타일 속성 : 원하는 값
        }

2.태그 선택자 (태그명)

해당 문서내에 같은 태그들 모두 선택할때 사용 

        태그명{
            스타일 속성 : 값;

        }

아이디 선택자(#아이디명)

해당 문서내에 특정 html요소 하나만을 선택하고자 할때 사용 단, 해당 요소 id속성을 이용해서 고유한 아이디값으 부여해야함 

            #아이디명{
                스타일속성 : 값
            }

클래스 선택자(.클래스명)

해당문서내에 내가원하는 여러개의 요소를 선택하고자 할때 사용. 

    .클래스명{
        스타일명 : 속성값
    }
