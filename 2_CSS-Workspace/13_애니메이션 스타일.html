애니메이션

속성을 점차적으로 변화시켜 움직이느 ㄴ효과를 내는 기법. 트랜지션과 비교했을때 시작스타일과 끝 스타일을 부드럽게 이어주는 기능을 제공함 애니메이션 중간에 원하는 위치에서 @keyfreams을 이용하여 중간 스타일 지정가능.

keyfreams

애니메이션 시작지점과 끝지점 스타일을 정하고 keyframes의 이름을 지정 시작지점과 끝지점 두개만 설정할 경우 from to 로 설정 중간에 작업이 더잇으면 %로 구분하여 설정. 

        @keyfreams 키프레임명 {
            선택자{스타일;}
