1. 속성 선택자

스타일을 부여하고자하는 요소내에 작성되어있는 속성을 이용해서 선택하는 방법. 선택자 뒤에 []를 이용해서 속성과 속성값을 재시해서 선택하는 방법. 

        선택자[속성=속성값]{} 속성값이 일치하는경우
        선택자[속성~=속성값]{} (공백,_,-)로 구분된 독립된 단어가 포함되어있는 경우
        선택자[속성|=속성값]{} 속성값이 포함되어있는경우
        선택자[속성*=속성값]{}
        선택자[속성^=속성값]{}

        a+b{스퇄}
        a요소 뒤에있는 모든 요소 b을 선택
        a~b{스타일}

4. 반응선택자

사용자의 움직임에 따라 선택이 되거나 안되는 선택자 

        *해당요소가 클릭 되었을 경우마ㅣㄴ 스타일 을 부여함
        선택자 : active {
            스타일:값
        }
        *해당 요소에 마우스가 올라갔을경우에만 스타일을 부여함
        선택자 : hover{
            스타일
        }

5. 상태선택자

요소의 상태에 따라서 선택되는 선택자. 

        1)checked 상태요소
        선택자:checked{
            스타일;
        }
    


        2) 초점(focus)이 맞춰진 요소 선택
아이디 : 

        3)활성화 되어잇는 요소들 선택
        선택자:enable{
            스타일
        }
        4)비활성화 되어잇는 요소들 선택
        선택자:disabled{
            스타일
        }
