form 태그내의 submit버튼 클릭시 해당 form내에 작성된
사용자가 입력한 값들을 서버로 넘기면서 요청하는 역할 수행
from action = "" method=""
* action 속성 : 해당 from내에 사용자가 입력한 값들을 전달하면서 요청할 서버의 경로 제시
* method 속성 : 요청 전송 방식을 지정하는 속성(get(기본)/post)
>get방식 : 요청시 사용자가 입력한 값들이 url에 노출되는 방식
>post방식 : 요청시 사용자가 입력한 값들이 url에 노출이 안되는 방식

from태그내의 제출(submit)버튼 클릭시 action에 지정한 서버로
method에 작성한 요청방식으로 사용자가 입력한 값들을 key=value세트로 전달함

select 태그와 option태그

-사용자에게 직접 키보드로 값을 입력받는게 아니라 내가 제시한 옵션들중에서 선택할수있게끔 해주는 태그
-제출시 현재 선택된(select)값이 넘어감 -각가의 option에 value 속성값을 명시했을경우 value값이 전달됨
-만일, 명시 안했을 경우 기본적으로 option태그의 content영역의 값이 넘어감.
